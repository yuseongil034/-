# 전자기학 벡터 해석 - 전자공학과 2학년용

## 📐 1. 좌표계 이해하기

### 직교 좌표계 (Cartesian Coordinate System)
우리가 가장 익숙한 x, y, z 축을 사용하는 좌표계입니다.

- **위치**: P(x, y, z) - 예: P(3, 4, 5)는 x축으로 3, y축으로 4, z축으로 5만큼 간 점
- **단위 벡터**: x̂, ŷ, ẑ (각 축 방향으로 크기가 1인 벡터)
- **미소 부피**: dV = dx dy dz

### 원통 좌표계 (Cylindrical Coordinate System)
원기둥 모양의 문제를 풀 때 편리합니다. (예: 동축 케이블, 원형 도선)

- **ρ (로)**: xy 평면에서 원점까지의 거리
- **φ (파이)**: x축에서 반시계 방향 각도
- **z**: 높이 (직교 좌표계와 동일)
- **미소 부피**: dV = ρ dρ dφ dz

### 구 좌표계 (Spherical Coordinate System)
구 모양의 문제를 풀 때 편리합니다. (예: 점전하, 구형 도체)

- **r**: 원점에서의 거리
- **θ (세타)**: z축에서 내려온 각도 (0° ~ 180°)
- **φ (파이)**: xy 평면에서의 방위각 (0° ~ 360°)
- **미소 부피**: dV = r² sin θ dr dθ dφ

## 🔄 2. 좌표 변환

### 직교 좌표 ↔ 원통 좌표

**직교 → 원통**
```
ρ = √(x² + y²)
φ = tan⁻¹(y/x)
z = z
```

**원통 → 직교**
```
x = ρ cos φ
y = ρ sin φ
z = z
```

**단위 벡터 변환 (암기 팁: cos-sin 행렬)**
```
ρ̂ = cos φ x̂ + sin φ ŷ
φ̂ = -sin φ x̂ + cos φ ŷ
ẑ = ẑ
```

### 직교 좌표 ↔ 구 좌표

**직교 → 구**
```
r = √(x² + y² + z²)
θ = cos⁻¹(z/r)
φ = tan⁻¹(y/x)
```

**구 → 직교**
```
x = r sin θ cos φ
y = r sin θ sin φ
z = r cos θ
```

## 📊 3. 벡터 미분 연산자

### 델(Del) 연산자: ∇
벡터 미분을 나타내는 기호로, 세 가지 방법으로 사용됩니다.

### 3-1. 기울기 (Gradient): ∇f
**의미**: 스칼라 함수 f가 가장 빠르게 증가하는 방향과 그 변화율

| 좌표계 | 공식 | 설명 |
|--------|------|------|
| **직교** | ∇f = ∂f/∂x x̂ + ∂f/∂y ŷ + ∂f/∂z ẑ | 각 축 방향의 변화율 |
| **원통** | ∇f = ∂f/∂ρ ρ̂ + (1/ρ)∂f/∂φ φ̂ + ∂f/∂z ẑ | φ 방향은 ρ로 나눔 |
| **구** | ∇f = ∂f/∂r r̂ + (1/r)∂f/∂θ θ̂ + (1/r sin θ)∂f/∂φ φ̂ | 각도 방향은 거리로 나눔 |

**예제**: 전위 V = x² + y² 일 때
- ∇V = 2x x̂ + 2y ŷ
- 점 (1,1,0)에서 ∇V = 2x̂ + 2ŷ (북동쪽 방향)

### 3-2. 발산 (Divergence): ∇·A
**의미**: 벡터장이 한 점에서 퍼져나가는 정도 (양수: 소스, 음수: 싱크)

| 좌표계 | 공식 | 외우는 법 |
|--------|------|-----------|
| **직교** | ∇·A = ∂Ax/∂x + ∂Ay/∂y + ∂Az/∂z | 각 성분을 각자의 변수로 미분 |
| **원통** | ∇·A = (1/ρ)∂(ρAρ)/∂ρ + (1/ρ)∂Aφ/∂φ + ∂Az/∂z | ρ 성분은 ρ를 곱한 후 미분 |
| **구** | ∇·A = (1/r²)∂(r²Ar)/∂r + (1/r sin θ)∂(sin θ Aθ)/∂θ + (1/r sin θ)∂Aφ/∂φ | r 성분은 r²를 곱한 후 미분 |

**예제**: 전기장 E = r̂/r² (점전하의 전기장)
- 구좌표에서 ∇·E = (1/r²)∂(r² × 1/r²)/∂r = 0 (원점 제외)

### 3-3. 회전 (Curl): ∇×A
**의미**: 벡터장이 한 점 주위에서 회전하는 정도

**직교 좌표계 (행렬식으로 계산)**
```
∇×A = | x̂    ŷ    ẑ   |
       | ∂/∂x ∂/∂y ∂/∂z |
       | Ax   Ay   Az   |

     = (∂Az/∂y - ∂Ay/∂z)x̂ + (∂Ax/∂z - ∂Az/∂x)ŷ + (∂Ay/∂x - ∂Ax/∂y)ẑ
```

**원통 좌표계**
```
∇×A = [(1/ρ)∂Az/∂φ - ∂Aφ/∂z]ρ̂ + [∂Aρ/∂z - ∂Az/∂ρ]φ̂ + [1/ρ(∂(ρAφ)/∂ρ - ∂Aρ/∂φ)]ẑ
```

**구 좌표계**
```
∇×A = 1/(r sin θ)[∂(Aφ sin θ)/∂θ - ∂Aθ/∂φ]r̂ 
     + 1/r[1/sin θ ∂Ar/∂φ - ∂(rAφ)/∂r]θ̂ 
     + 1/r[∂(rAθ)/∂r - ∂Ar/∂θ]φ̂
```

### 3-4. 라플라시안 (Laplacian): ∇²f = ∇·(∇f)
**의미**: 함수의 곡률, 주변 평균값과의 차이

| 좌표계 | 공식 | 물리적 의미 |
|--------|------|-------------|
| **직교** | ∇²f = ∂²f/∂x² + ∂²f/∂y² + ∂²f/∂z² | 각 방향의 2차 미분 합 |
| **원통** | ∇²f = (1/ρ)∂/∂ρ(ρ ∂f/∂ρ) + (1/ρ²)∂²f/∂φ² + ∂²f/∂z² | 원통 대칭 고려 |
| **구** | ∇²f = (1/r²)∂/∂r(r² ∂f/∂r) + (1/r² sin θ)∂/∂θ(sin θ ∂f/∂θ) + (1/r² sin² θ)∂²f/∂φ² | 구 대칭 고려 |

## 🔌 4. 전자기학 핵심 방정식

### 맥스웰 방정식

| 법칙 | 미분 형태 | 의미 | 예시 |
|------|-----------|------|------|
| **가우스 법칙** | ∇·E = ρ/ε₀ | 전하가 전기장의 원천 | 점전하 주위의 전기장 |
| **자기 가우스 법칙** | ∇·B = 0 | 자기 홀극 없음 | 자기력선은 항상 닫힌 곡선 |
| **패러데이 법칙** | ∇×E = -∂B/∂t | 변하는 자기장이 전기장 유도 | 변압기 원리 |
| **암페어-맥스웰 법칙** | ∇×B = μ₀J + μ₀ε₀∂E/∂t | 전류와 변하는 전기장이 자기장 생성 | 전자석, 전자기파 |

### 전위와 전기장
```
E = -∇V  (전기장은 전위가 감소하는 방향)
```

### 포아송/라플라스 방정식
```
∇²V = -ρ/ε₀  (포아송 방정식: 전하가 있는 곳)
∇²V = 0      (라플라스 방정식: 전하가 없는 곳)
```

## 💡 5. 문제 풀이 팁

### 좌표계 선택 가이드
1. **직교 좌표계**: 박스 모양, 평면 문제
2. **원통 좌표계**: 원기둥, 동축 케이블, 무한 직선 전하
3. **구 좌표계**: 점전하, 구형 도체, 구면파

### 암기 팁
1. **발산**: "들어오고 나가는 것의 차이"
2. **회전**: "오른손 법칙으로 회전 방향 결정"
3. **기울기**: "가장 가파른 오르막 방향"

### 자주 나오는 결과
1. 점전하의 전기장: E = (1/4πε₀) × (q/r²) r̂
2. 무한 직선 전하의 전기장: E = (λ/2πε₀ρ) ρ̂
3. 무한 평면 전하의 전기장: E = (σ/2ε₀) n̂

## 📝 6. 벡터 항등식 (시험에 자주 나옴)

| 항등식 | 의미 |
|--------|------|
| ∇·(∇×A) = 0 | 회전의 발산은 항상 0 |
| ∇×(∇f) = 0 | 기울기의 회전은 항상 0 |
| ∇(fg) = f∇g + g∇f | 곱의 기울기 |
| ∇·(fA) = f(∇·A) + A·(∇f) | 곱의 발산 |
| ∇×(fA) = f(∇×A) - A×(∇f) | 곱의 회전 |
